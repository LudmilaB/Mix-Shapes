Fit Shapes

Fit as many shapes as you can

0

HOW TO PLAY: Touch empty space to grow a new shape. When a new shape touches an existing shape of same color, they both disappear. If they have different colors, both stay. If the new shape cannot fit, a minimal dotted outline is shown. After 3 such misfits in a row, the game is over. If you touch inside a standalone shape, it will grow more.
